# 📦 길음중 아나바다 장터 (GRMS-anabada) - 과제 제출용 복제본

> 본 저장소는 교육용으로 제작된 **'길음중 아나바다 장터'** 애플리케이션의 **과제 제출용 복제본(Replica)**입니다. 실제 상용/운영 목적의 메인 저장소가 아니며, 제출 및 검토를 위해 코드를 정리하여 재구성하였습니다.

---

## 🌟 프로젝트 개요
**길음중 아나바다 장터(GRMS-anabada)**는 학교 내에서 학생들이 직접 환경 보호와 자원 순환의 가치를 배우고 실천할 수 있도록 돕는 **교육용 모의 마켓 시스템**입니다. 
가상의 화폐(G포인트)와 환경기여도(리프) 개념을 도입하여, 안전한 가상 공간 속에서 실감 나는 아나바다 장터 거래를 경험할 수 있습니다.

---

## 🔒 개인정보 보호 및 보안 특장점
학교 및 교육 현장에서 학생들이 안전하게 사용할 수 있도록 **설계 단계부터 개인정보 수집을 원천 차단**하였습니다.
- **가상 계정 체계**: 실명, 이메일, 전화번호 등 실제 개인정보는 일절 수집하지 않으며, 임의 생성된 랜덤 아이디 및 학년·반·번호로만 계정이 구성됩니다.
- **안전한 데이터 관리**: 비밀번호는 단방향 해시(Hash) 암호화되어 저장되므로 관리자나 개발자도 식별할 수 없습니다.
- **행사 후 즉시 폐기**: 수집된 모의 데이터는 행사 종료(2026년 7월 20일) 즉시 완전 삭제(Hard Delete)됩니다.

## 🛠 기술 스택
- **Frontend**: Vanilla HTML5, Vanilla CSS3, Vanilla JavaScript (ES6+ Module)
- **Backend / Database**: Google Firebase (Firestore Database, Firebase Auth)
- **Deployment**: Vercel
- **Version Control**: GitHub

---

## ✨ 핵심 기능
1. **사용자 역할별 대시보드**
   - **일반 학생**: 내 보유 G포인트, 환경 리프 점수, 거래 건수 실시간 조회 및 모의 거래 기능.
   - **상인 학생 (학급 대표)**: 학급 물품 마켓 지정(A/B 그룹 분할) 및 판매 완료 처리 기능.
   - **담임 교사 (VIP)**: 소속 학급 학생들을 상인으로 지정하거나 해제하는 관리 권한.
   - **전체 관리자 (Admin)**: 전체 이용자 역할 부여, 카테고리별 환경 리프 점수 설정, 체크리스트 관리, 마켓 단계 제어(개장 전/A마켓/B마켓/마감) 및 통계 데이터 다운로드.

2. **환경 성장 비주얼라이징**
   - 학급별 누적 획득 리프(환경 점수)에 따라 자라나는 가상의 환경 나무(Tree) 화면 제공 (전자칠판 전시용).

3. **물품 및 모의 거래 시스템**
   - 찜하기 기능, 구매 내역 확인, 모의 바코드/물품 번호를 이용한 상인의 실시간 판매 승인 처리.
